# Link Devices to Arrangements and Retain Account Resolution

A Payment Device Link permanently targets the Product Arrangement selected when the Link is created rather than one Ledger Account. For each accepted operation, the rules of the Arrangement's bound Product Definition resolve its purpose and Asset to one assigned Ledger Account, and the operation owner retains that exact resolution for delayed handling and retries. Reassignment still requires deactivating the old Link and creating a new one, while retaining Account resolution prevents later Assignment changes from reinterpreting accepted work.
